Einstellungen
Die Einstellungen dienen überwiegend zur Verwaltung des RoomPlaners. Hier können neue Benutzer angelegt und vorhandene bearbeitet oder gelöscht werden.

Benutzer
Über den Punkt "Benutzer" können die in der Anwendung vorhandenen Benutzer betrachet, editiert und gelöscht, sowie neue Benutzer angelegt werden.
Sollte nur ein Administrator im System vorhanden sein, kann dieser weder gelöscht werden, noch seine Administrator-Rechte verlieren.

Folgende Regelungen Gelten für den Benutzer-Bereich
  • Neue Benutzer müssen nach der Erstanmeldung ihr Kennwort ändern.
  • Ist nur ein Benutzer mit Administrator-Rechten vorhanden, kann er diese nicht verlieren und auch nicht gelöscht werden
  • Hat ein Benutzer keine Administrator-Rechte, kann er den Bereich "Einstellungen" nicht sehen.

Info
Im Bereich "Info" werden Informationen zur Software und dessen Version bereitgestellt.
