Belegungs-Plan
Damit der Belegungs-Plan korrekt genutzt werden kann, muss zunächst mindestens ein Zimmer und ein Gast existieren.
Zimmer können über die Zimmer-Verwaltung angelegt werden. Gäste können direkt beim Anlegen einer Belegung oder über die Gäste-Verwaltung erstellt werden.

Neue Belegung anlegen
Für das Anlegen einer neuen Belegung muss das Formular im unteren Bereich der Seite vollständig ausgefüllt werden.
  • Datum von
  • Datum bis
  • Zimmer auswählen
  • Vorhandenen Gast suchen (sollte der gesuchte Gast nicht vorhanden sein, kann er neu angelegt werden)
  • Bemerkung (optional)
Anschließend muss die neue Belegung abgespeichert werden. Sollten Felder nicht oder unlogisch befüllt worden sein, weißt das Formular darauf hin.

Vorhande Belegung editieren
Um eine vorhandene Belegung zu editieren, muss sie über den Kalender angewählt werden.
Das Formular im unteren Bereich der Seite wird dann mit den vorhandenen Daten gefüllt und kann anschließend editiert werden.

Vorhande Belegung löschen
Soll eine vorhandene Belegung gelöscht werden, muss sie über den Kalender angewählt werden.
Über das Formular im unteren Bereich der Seite, kann nun die Belegung gelöscht werden.
