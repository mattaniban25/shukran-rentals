Gäste-Verwaltung
Die Gäste-Verwaltung kann neue Gäste anlegen und vorhandene editieren oder löschen.
Es können mehrere Aktionen gleichzeitig ausgeführt werden (z.B. neuen Gast anlegen, vorhandenen Gast editieren und ein vorhandenen Gast zum Löschen vormerken). Die Änderungen werden aber erst nach dem Speichern wirksam.

Vorhandene Gäste werden im Belegungs-Plan zur Erstellung neuer Belegungen verwendet.

Neuen Gast erstellen
Neue Gäste werden über das Plus oben rechts in der Tabelle hinzugefügt, werden der Tabelle angefügt und grün hinterlegt.
Damit neue Gäste abgespeichert werden können, müssen alle Felder gefüllt sein.
  • Name
  • Vorname
  • Adresse
Neue Gäste werden erst abgespeichert, wenn der Button "Speichern" gedrückt wird und sind anschließend nicht mehr grün hinterlegt. Die Erstellung kann durch das "X" am Ende der Zeile abgebrochen werden. Die Zeile wird wieder entfernt.

Das Feld "Belegungen" wird automatisch durch das Zählen der vorhandenen Belegungen für den Gast gefüllt.

Vorhandenen Gast bearbeiten
Die Werte eines Gastes können direkt in der Tabelle editiert werden.
Änderungen werden erst durch einen Klick auf den Button "Speichern" wirksam.

Vorhandenen Gast löschen
Ein Gast kann durch das Klicken auf das "X" am Ende der Zeile des jeweiligen Gastes zum Löschen markiert werden. Der / die Gäste werden erst gelöscht, wenn der Button "Speichern" geklickt wird.
